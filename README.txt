ODOO SALES SYNC MODULE
Version: 1.0.0
PrestaShop: 8.0+

INSTALLATION:
1. Upload ZIP via Back Office > Modules > Upload a module
   OR
2. Extract to modules/odoo_sales_sync/ and install from Module Manager

CONFIGURATION:
After installation:
1. Go to Modules > Odoo Sales Sync > Configure
2. Set Webhook URL (your Odoo endpoint)
3. Set Webhook Secret (for authentication)
4. Enable Sync
5. Save
